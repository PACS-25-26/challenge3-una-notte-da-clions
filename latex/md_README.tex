This project implements a solver suite for elliptic boundary value problems on a square domain, with support for\+:


\begin{DoxyItemize}
\item {\bfseries{Dirichlet}}, {\bfseries{Neumann}}, and {\bfseries{Robin}} boundary conditions
\item iterative solvers {\bfseries{Jacobi}} and {\bfseries{Schwarz}}

\item verification tests with analytical solutions and convergence monitoring
